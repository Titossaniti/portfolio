Ce portfolio a été codé en HTML et CSS et contient un peu de JavaScript pour les animations et les intégrations.

Sont intégrés en JS plusieurs éléments, de façon dynamique de sorte que les liens ne soient pas obsolètes selon la page sur laquelle on se trouve. Celà concerne la navbar (le menu tout en haut de la page pour naviguer sur le site), le footer, ainsi que les liens vers mes réseaux sociaux professionnels, qui se trouvent sur le côté gauche de la page.

Le frameworks CSS utilisé est Bootstrap. Concernant le JavaScript, jQuery a servis de bibliothèque, bien que peut utilisé.

SaticForms me fournis les données envoyées via le formulaire de contact.
